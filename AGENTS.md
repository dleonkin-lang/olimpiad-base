# AGENTS.md — правила работы с проектом

Проект: база олимпиадных задач 5–7 класса в статическом HTML/JSON.

## Жёсткие правила

1. Не менять рубрики, темы, подтемы, методы и интерфейс без отдельной явной просьбы.
2. Перед крупными изменениями сохранить текущую версию как точку отката.
3. Если импортируются новые задачи, добавить новый источник, если его ещё нет.
4. Если в PDF есть решения, добавлять ответы и подробные решения.
5. Рубрикацию ставить с учётом решения, если решение есть.
6. В конце любой рабочей итерации выдавать: рабочий HTML, публичный read-only HTML, JSON базы задач, полный JSON-бэкап, каталог источников и архив всей версии.
7. `docs/index.html` должен оставаться read-only: `CAN_EDIT = false`.
8. `docs/admin.html` должен оставаться рабочей версией: `CAN_EDIT = true`.
9. Сейчас в базе 237 задач и 51 источник. Нельзя терять существующие задачи и источники.

## Первый проход в Codex

Первый запрос к Codex должен быть только разведочным: оценить структуру проекта, проверить как всё устроено, но ничего не менять.
